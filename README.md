# 지브리 open API 연동

## 프로젝트
vue 를 이용한 지브리 영화 소개리스트 <br>
vuex 를 이용해 store 에서 컴포넌트로 데이터 참조 <br>
vue router 를 이용해 페이지이동 구현

#### 사용기술 : Vue, JavaScript, CSS

##### 라이브러리 : axios, Vuex, Vue-router
